# 🌟 JoJo Star Gerador de Marcas Estelares

O **jojo star** é uma ferramenta autoral de design generativo que permite criar, personalizar e catalogar formas estelares complexas. Originalmente inspirado na estética de "JoJo's Bizarre Adventure", o projeto evoluiu de um simples gerador para uma plataforma com persistência de dados em nuvem.

## 🚀 Diferenciais Competitivos (Fase 2)
Para esta etapa de inovação, foram implementados 4 recursos fundamentais utilizando o ecossistema Firebase:

1.  **Stellar Auth (Firebase Authentication):** Sistema de login seguro via Google, permitindo que cada usuário tenha sua própria "linhagem" de estrelas.
2.  **Private Nebula (Cloud Firestore):** Persistência de dados para salvar configurações personalizadas (pontas, brilho, cor, velocidade e 3D).
3.  **Galeria Dinâmica:** Painel lateral para gerenciamento, visualização e recuperação instantânea de designs salvos.
4.  **Interface Adaptativa:** Design moderno com suporte a Dark/Light mode e tradução completa para Português.

## 🛠️ Tecnologias Utilizadas
- **React + TypeScript**
- **Tailwind CSS** (Estilização)
- **Lucide-React** (Ícones)
- **Framer Motion** (Animações e 3D)
- **Firebase** (Auth e Firestore)

## 📦 Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure suas chaves do Firebase no arquivo de configuração.
4. Inicie o servidor: `npm run dev`.
